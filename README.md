## Prerequisites
- [hugo](https://gohugo.io/)
## Usage
### clone theme
```
git submodule update --init --recursive
```
#### Create new content
```
hugo new content <SECTIONNAME>/<FILENAME>.<FORMAT>
```
#### Run development server
```
make dev
```
#### Update theme
```
make update
```
